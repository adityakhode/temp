Drone in a Box ​

Overview ​

"Drone in a Box" is a comprehensive solution designed to provide advanced drone capabilities through a convenient, all-in-one package. This package includes three cutting-edge drones: the Swift Pro, Swift Pro-Max, and Holybro X500. Each drone is tailored to meet various needs, from professional photography to complex surveying tasks.

Included Drones ​

  1. Swift Pro: A versatile drone suitable for a range of applications with impressive performance and payload capacity.
  2. Swift Pro-Max: An advanced version of the Pro with enhanced features and extended capabilities.
  3. Holybro X500: A high-performance drone designed for heavy-duty applications and complex missions.

For detailed information about each drone, please refer to the individual README files:

  • Swift Pro Drone
  • Swift Pro-Max Drone
  • Holybro X500 Drone
